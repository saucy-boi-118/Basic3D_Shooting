# Basic3D_Shooting
A Basic shooting and collision system frame in C# Raylib
# File Structure
The Basic 3D is sorted into different versions ranging from 1.0 to 3.0 where 3.0 is the most recent final version. Click on the folder to access the code or ZIP file. Built in C# Raylib with little to no help from AI sources. Collision system is spatial partitioning and is really simple.
